awwwards-3d

A Claude Code skill for building Awwwards-style scroll-driven 3D websites. Three.js r170 + GSAP 3.12.5 + Lenis 1.1 + three's built-in EffectComposer. Four verified templates, 33 documented patterns, and Blender MCP recipes that actually run end-to-end. View on GitHub →

01 — Foundation
minimal.html
Empty scene with the canonical lerp + scroll + composer chain wired up. Chrome icosahedron, ACES tone mapping, bloom, vignette, grain.
02 — Object showcase
coin-scroll.html
Gold coin tumbles + falls + drifts on scroll. Loads coin.glb via DRACOLoader (6 KB Draco-compressed from Blender).
03 — Spline camera
room-walkthrough.html
Camera follows a CatmullRomCurve3 path through a low-poly interior. Painting + hero orb + warm lamp + cool window light.
04 — Glass
glass-product.html
Hero orb with MeshPhysicalMaterial transmission + dispersion (r166+ chromatic refraction). No drei dependency.
